# M1-S3-Advanced-Inventory-Management-with-Collections-and-File-Persistence


## What is Advanced-Inventory-Management-with-Collections-and-File-Persistence?
It is a Python inventory system that:
- Runs from the command line
- Uses lists, dictionaries, and functions
- Is organized into modules
- Allows saving and loading data to CSV
- Calculates statistics and evaluates validations

## Main Features:
Allows you to:
- Add products
- View inventory
- Search for products
- Update products
- Delete products
- View statistics
- Save to CSV
- Load from CSV
- Exit
>All of this from an interactive menu (1–9)
